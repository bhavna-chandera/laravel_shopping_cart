# laravel_shopping_cart

A full-featured web application built with Laravel that includes both User Panel and Admin Panel for managing the system efficiently.

---
## 🛠 Tech Stack

- Backend: Laravel
- Frontend: Blade
- Database: MySQL
- Authentication: Laravel Auth Breeze
- Server: Apache / XAMPP
